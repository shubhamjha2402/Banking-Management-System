Steps to run the program:
Step 1: Run the server.c file which will create the banking server.Once the server side is set, we can run the client program.
Step 2: Run the client.c file on different terminal which will prompt different account options for the user. The admin can be created using database.c 
Upon accessing the admin, it can add or remove the account of customers.
There are two types of customer: Single and joint account holders. Both of them get options related to their account.
